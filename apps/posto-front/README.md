# posto-front — App do Frentista

App que roda na Smart Cielo do posto.

- Recebe notificações de abastecimento
- Aprova ou recusa via terminal
- Monitora bombas em andamento
- Impressão automática de comprovante

**Status:** a ser desenvolvido
**Plataforma:** React Native (Android — Smart Cielo)
**Deploy:** via MDM da Cielo
